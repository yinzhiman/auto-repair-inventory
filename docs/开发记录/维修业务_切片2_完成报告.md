## 问题与决策

### 测试数据隔离问题

**情况**：原测试使用 beforeEach 清空数据，导致后续测试找不到数据
**处理**：移除 beforeEach，改为在每个测试中动态获取当前状态
**影响**：测试更稳定，但测试间有依赖

### 前端编辑功能集成

**情况**：Task-04 页面已包含编辑功能，Task-06 无需新增页面
**处理**：直接复用 Task-04 的工单表单组件
**影响**：减少重复代码，功能一致

## 技术债务

- **已完成工单编辑**：后端 API 支持修改任意状态工单，但前端只允许修改 pending 状态。需要确认业务需求后决定是否开放。

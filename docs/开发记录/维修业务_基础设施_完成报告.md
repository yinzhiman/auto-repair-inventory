## 问题与决策

### TypeScript 类型断言问题

**情况**：测试文件中 `loginRes.headers['set-cookie']` 返回类型为 `string`，但实际是 `string[]`
**处理**：使用 `as unknown as string[]` 双重断言
**影响**：无，仅影响测试代码

### 测试数据清理顺序

**情况**：删除测试数据时外键约束报错
**处理**：调整删除顺序为 stockLog → orderItem → order → part → vehicle → customer
**影响**：无，测试运行正常
